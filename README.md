# Rainwater Harvesting Website

This is an **informative website** about rainwater harvesting, created as a beginner project to showcase basic web development skills.

## Features
- Simple and clean **landing page** layout
- Informative content about rainwater harvesting techniques
- Responsive design using **HTML & CSS**
- Easy to navigate sections

## Skills / Tools Used
- HTML
- CSS
- Basic JavaScript (if any)
- GitHub for version control

## Instructions
- Open `index.html` in a browser to view the website
- All assets (images, CSS) are included in the repository

> This project is a **practice / learning project** created by a beginner developer.
